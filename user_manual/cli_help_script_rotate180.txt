User Manual
===========

Page: 'rotate180'
Category: 'script command'

Goal:
-----
Rotate the input image 180 degree clockwise.

Syntax:
-------
`rotate180`

Usage example:
--------------
`sic input.png output.png --script "rotate180;"`
