User Manual
===========

Page: 'rotate270'
Category: 'script command'

Goal:
-----
Rotate the input image 270 degree clockwise.

Syntax:
-------
`rotate270`

Usage example:
--------------
`sic input.png output.png --script "rotate270;"`

---------
