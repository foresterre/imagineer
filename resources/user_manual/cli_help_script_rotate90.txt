User Manual
===========

Page: 'rotate90'
Category: 'script command'

Goal:
-----
Rotate the input image 90 degree clockwise.

Syntax:
-------
`rotate90`

Usage example:
--------------
`sic input.png output.png --script "rotate90;"`

---------
